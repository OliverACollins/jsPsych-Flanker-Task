# jsPsych-Flanker-Task
<b>jsPsych implementation of a basic flanker task.</b>

[Try the implementation](https://oliveracollins.github.io/jsPsych-Flanker-Task)

## The Task
- Includes **congruent** and **incongruent** stimuli (NOT **neutral** stimuli). There are 2x types of congruent and incongruent stimuli, respectively
- Each trial is preceded by a fixation cross lasting between 500 and 1000 ms
- Each stimulus lasts for a maximum of 2000 ms
- Practice block of 16 trials
- Experimental trials: separated into 4x blocks of 40 trials each (160 total trials)
- Feedback after each block: inverse efficiency score (IES), correct % of responses, and average response time (RT)
